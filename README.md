explicacion corta:
el programa tiene funciones de una biblioteca. La idea es tener unos libros preparados. Esos libros no son reales, sino que se crean mezclando frases al azar con los géneros principales, así siempre hay algo disponible para mostrar, aunque el usuario no haya registrado nada todavía. 

se puede registrar libros. El programa te pregunta cuántos quieres añadir, luego te pide uno por uno el nombre y al final te dice que elijas a qué género pertenece. Si no encuentras tu género ahí, también puedes escribir el que quieras.

También está la función que  deja mirar todo lo que hay disponible. Ahí es donde se muestran los libros que trae el sistema de manera predeterminada y los se que se han agregado. Si no hay libros registrados el programa avisa 

Y finalmente, el programa da opción de mirar por categorías. Es decir, si se desea mirar comedia o solo romance ejemplo, El sistema va preguntando si el usuario quiere seguir mirando o si ya acabo. 
se utilizan varios conceptos como clases, heriencia, se realiza un menu general donde ocurre el menu, se utilizan metodos privados en el nombre
